# Android Studio汉化包（需手动安装）
## 注意:本项目只测试Mac arm版本，Intel版本自行尝试

1、首先前往【关于AndroidStudio】查看具体版本号
<img width="1048" height="616" alt="image" src="https://github.com/user-attachments/assets/ce12f10d-19e6-4ffa-97ab-edaf3ded20cd" />

<img width="1048" height="616" alt="image" src="https://github.com/user-attachments/assets/528ddb65-3e9b-43d7-b7d3-58db0ad46e32" />

2、下载对应的版本号汉化包，前往AndroidStudio【插件】->【设置】->【从磁盘安装插件】，选择对应的zip文件，进行安装（没有的可以提需求，我来补充）
<img width="2114" height="1314" alt="image" src="https://github.com/user-attachments/assets/473266fa-04a3-434b-859e-8ffb6591224c" />

<img width="2114" height="1314" alt="image" src="https://github.com/user-attachments/assets/8af96e82-4bbd-434e-ad0d-be6d4fc5d9cb" />

3、重启，设置 | 外观与行为 | 系统设置 | 语言和区域  选择【简体中文】大功告成！

<img width="1964" height="1444" alt="image" src="https://github.com/user-attachments/assets/29b901e9-e392-4be5-95a3-26a9b29f1f12" />
